- Microsoft powerPoint
- [**Anaconda**](https://www.anaconda.com/)
- [**Julia**](https://julialang.org/)
- [**Mathematica**](https://www.wolfram.com/mathematica/)

(→ 학기가 다 끝나고 보니, 대부분은 Anaconda 설치시 같이 설치되는 Python 3을 주로 사용했습니다.)
